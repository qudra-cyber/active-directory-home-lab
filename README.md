



# Active Directory Home Lab

## Overview

This project demonstrates the deployment and administration of Microsoft Active Directory in a Windows Server environment.

The lab simulates a small company by creating Organizational Units (OUs), security groups, and user accounts for multiple departments.

## Technologies Used

* Windows Server 2022
* Active Directory Domain Services (AD DS)
* VirtualBox
* Windows Administration

## Domain

qudratech.local

## Organizational Units

* HR
* IT
* Finance
* Sales

## Security Groups

* HR_GROUP
* IT_GROUP
* IT_MANAGERS
* FINANCE_GROUP
* SALES_GROUP

## User Administration

Created departmental user accounts and assigned them to appropriate security groups.

Examples:

* Benjamin IT
* Juma IT
* Sarah HR
* John HR
* David Finance
* Grace Sales

## Skills Demonstrated

* Active Directory Administration
* User Provisioning
* Security Group Management
* Organizational Unit Management
* Identity Administration
* Windows Server Administration
